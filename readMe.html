Read Me: Inventory Management System

This system is meant for inventory adminstrators that wants to keep track of the company's inventory to stay organized and use it as a point of reference to go through their catalog.

Stored Procedures Used

  • updateProduct: updates a product with respect to what is changed by the user.
  • getProductList: fetches all the products name with their id to be used by the list box
  • getDepartmentList: fetches all the departments name with their id to be used by the list box
  • getSupplierList: fetches all the products name with their id to be used by the list box.
